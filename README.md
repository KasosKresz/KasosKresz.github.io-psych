# Upload-Ready Website

This folder contains the version to publish.

What to upload:
- Upload the contents of this folder to the root of your GitHub repository.
- Do not upload the `upload-ready` folder itself as a subfolder.

What is included:
- `index.html` as the root language-choice page
- `en/index.html` for English
- `hu/index.html` for Hungarian
- `profile-photo.png`
- `CNAME`
- `robots.txt`
- `sitemap.xml`

Resulting URLs:
- `https://krisztiankasos.mindease.ca/`
- `https://krisztiankasos.mindease.ca/en/`
- `https://krisztiankasos.mindease.ca/hu/`

GitHub Pages:
- Push these files to the branch GitHub Pages is serving.
- In GitHub `Settings` > `Pages`, publish from your main branch and root.
- Keep the custom domain set to `krisztiankasos.mindease.ca`.
